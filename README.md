# RoadWatch

RoadWatch is an AI-powered road safety and transparency platform developed for a road safety hackathon.

## Features

* Road condition monitoring
* Complaint reporting system
* Road safety dashboard
* AI assistant for citizen queries
* Infrastructure transparency tools
* Data-driven accident risk analysis

## Tech Stack

### Frontend

* React
* TypeScript
* Vite
* Tailwind CSS
* React Router

### Backend

* Python / FastAPI (planned)

### Database

* PostgreSQL (planned)

### Mapping

* OpenStreetMap
* Leaflet (planned)
